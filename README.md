
# Sistema Espaço Lição de Casa

## Resumo

Sistema de gestão para a escola de reforço Lição De Casa.

## Atores/Personas

Secretárias, Professores, Alunos e Proprietária da Instituição.

## Principais Dores

* Problemas ao gerenciar agenda de aulas
* Falta de organização e visualização das despesas da Instituição
* Dificuldade ao gerenciar valores a receber
* Dificuldade em gerenciar o cadastro de Alunos
